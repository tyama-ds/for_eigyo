Indexes — 検索対象の索引

「走査」で索引フォルダを検出します
＋ 索引を作成（フォルダ → 索引）
既存の索引名を指定すると追記になります（同名ファイルはスキップ）。 取り込みには埋め込みAPIが必要です。
✓ を付けた索引が横断検索の対象になります。
索引は PagedRAG / BookRAG の storage フォルダをそのまま検出します （同じ埋め込みモデルで作成されている必要があります）。
